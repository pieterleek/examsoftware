# Welkom bij de proeve van SD – Leerjaar 1

In deze 3 dagen ga je een **roomservice-bestelsite** bouwen voor het chique hotel Rits Karton Zaandam.  
Gasten kunnen via de site gerechten bestellen, het aantal kiezen, en hun kamernummer en gewenste bezorgtijd invullen.  

Je werkt in **drie losse opdrachten**, verdeeld over **drie dagdelen**.  
Elke opdracht bouwt voort op de vorige.

## 🔧 Aandachtspunten voordat we beginnen

- Je maakt gebruik van **PHP** en **MySQL** om bestellingen op te slaan.  
  Zorg dat je een **werkende XAMPP-omgeving** hebt.
- We werken met de **talland.cloud GitLab-server**.  
  Hier vind je een **startproject** dat je kunt gebruiken om snel te beginnen.  
  Je levert hier ook je eindproduct weer in.
